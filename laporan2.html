Laporan Konfigurasi MikroTik dengan WinBox

A. Apa itu WinBox?

WinBox adalah sebuah utilitas kecil yang memungkinkan administrasi MikroTik RouterOS menggunakan antarmuka grafis yang cepat dan sederhana. WinBox adalah aplikasi native untuk Windows (Win32/Win64), tetapi juga dapat dijalankan di Linux dan macOS menggunakan Wine. Dengan WinBox, kamu bisa mengelola berbagai fungsi router seperti routing, firewall, manajemen bandwidth, titik akses nirkabel, gateway hotspot, server VPN, dan banyak lagi. Semua fungsi antarmuka WinBox dirancang sedekat mungkin dengan fungsi konsol, sehingga memudahkan pengguna yang terbiasa dengan perintah konsol.

B. Topologi Jaringan

MikroTik adalah perangkat keras dan perangkat lunak yang digunakan untuk mengelola dan mengkonfigurasi jaringan. Salah satu fitur yang sering digunakan dalam konfigurasi MikroTik adalah bridge, yang memungkinkan penggabungan dua atau lebih antarmuka jaringan menjadi satu jaringan logis. Fitur ini berguna dalam berbagai situasi, seperti membangun jaringan lokal (LAN), jaringan nirkabel (Wi-Fi), atau mengimplementasikan fitur seperti VLAN.

Ada dua mode konfigurasi utama pada MikroTik: mode router dan mode bridge (switch).

  • Mode Router: Mode router adalah mode operasi standar dari MikroTik RouterOS. Dalam mode ini, perangkat MikroTik berfungsi sebagai router yang mengarahkan lalu lintas data antar jaringan yang berbeda.
  • Mode Bridge (Switch): Mode ini mengubah perangkat MikroTik menjadi switch yang menghubungkan beberapa perangkat dalam jaringan lokal (LAN) dan memungkinkan komunikasi antar perangkat tersebut di tingkat lokal.

C. Prosedur Mode Router

Berikut adalah langkah-langkah dasar untuk mengkonfigurasi router MikroTik RB931-2nD dengan IP 192.168.1.40/24:

  1. Hubungkan Router ke PC: Sambungkan router MikroTik ke PC menggunakan kabel Ethernet. Gunakan salah satu port dari 2 hingga 5.
  2. Unduh dan Buka WinBox: Unduh WinBox dari situs resmi MikroTik, buka WinBox, dan temukan alamat MAC router di tab “Neighbors”.
  3. Hapus Konfigurasi Default: Setelah login, klik “Remove Configuration” untuk memulai dari awal.
  4. Setel IP Address untuk LAN: Buka menu IP > Addresses dan tambahkan alamat IP 192.168.1.40/24.
  5. Konfigurasi DHCP Server: Buka menu IP > DHCP Server, tambahkan jaringan baru, dan atur gateway ke 192.168.1.40.
  6. Setel NAT untuk Koneksi Internet: Buka menu IP > Firewall dan tambahkan aturan NAT baru dengan menggunakan “masquerade”.
  7. Simpan dan Reboot: Setelah semua konfigurasi selesai, simpan perubahan dan reboot router.
